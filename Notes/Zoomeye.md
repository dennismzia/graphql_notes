# ZoomEye Search Terms Reference

## Basic Search Terms

1. **IP Address Search**:  
   `ip:192.168.1.1`  
   `ip:192.168.1.0/24`

2. **Port Search**:  
   `port:80`  
   `port:22`  
   `port:443`

3. **Service Search**:  
   `service:ftp`  
   `service:http`  
   `service:mongodb`  
   `service:mysql`  
   `service:rdp`

4. **Banner Grabbing**:  
   `banner:"Apache"`  
   `banner:"nginx"`  
   `banner:"Microsoft-IIS"`

5. **Operating System (OS) Search**:  
   `os:Windows`  
   `os:Linux`
